Aide de Gamebox

I. Commandes de jeu

1. Sur le plateau de jeu

  • Clic gauche : effectuer un mouvement.
  • Clic gauche (SHIFT enfoncé) : défaire le précédent mouvement.
  • Clic droit enfoncé : voir les mouvements légaux.

2. Dans « À propos de »

  • Clic gauche : faire tourner l'anneau sélectionné dans le sens horlogique.
  • Clic droit : faire tourner l'anneau sélectionné dans le sens anti-horlogique.
  • Clic gauche (SHIFT enfoncé) : faire tourner le disque dans le sens horlogique.
  • Clic droit (SHIFT enfoncé) : faire tourner le disque dans le sens anti-horlogique.

II. Statistiques

1. Comment calculer les stats pour gnuplot ?

Dans un terminal, lancez AiStats : java -cp gamebox.jar be.ac.umons.informatique.ba1.gamebox.ui.AiStats FIXME .

III. Compilation

1. Comment créer le fichier jar ?

Dans un terminal, tapez ant jar. Un fichier gamebox.jar (que vous pourrez ouvrir sans avoir besoin du JDK) sera généré...

2. Comment compiler depuis Eclipse ?

En règle générale, il faut créer un lien symbolique de /code/res vers /res.

Sous Windows, utilisez junction (cet outil fait partie des Windows Sysinternals) : junction code\other res.

Sous GNU/Linux, utilisez ln : ln -s ./res ./code/res.
